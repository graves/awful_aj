# Use `aj` 🚀

AJ can be used in three primary ways:

1. **Init**: Bootstrap local config/templates/DB 🏗️
2. **Ask**: One‑shot Q&A ✨
3. **Interactive**: Chat with memory 🧠
4. **As a Library**: Embed AJ in your Rust code 🧩
